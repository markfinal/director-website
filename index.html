Director

Director enables users to create animated CG scenes, play them back in realtime, or save a movie with frames rendered by PRMan.

Drag-n-drop geometry files onto the Director UI, and create instances of Actors using them.

Design shaders with the built-in node graph, using a subset of the Open Shading Language specification, and apply them to your Actors, to light and shade them how you imagined.

Animate virtually anything (Actors, Cameras, Lights, Shader parameters), by specifying property values at keyframes of your choice, in the Dope Sheet.

Preview in realtime on your GPU using OpenGL, for an approximate look of your animation.

Render individual frames using PRMan to check your look-dev.

Render your entire animation using PRMan, and create a movie file by combining the frame images using ffmpeg.

Share or collaborate your animations by saving them to the cloud.

Additional Requirements

To use PRMan requires a separate download and license.

ffmpeg used to create movie files requires a separate download.

Blender requires a separate download in order to load .blend files.

git is a separate download and may be required to upload projects to cloud based repositories.
